# 多平台支持说明

## 支持的平台

本项目现已支持以下平台：

### 🪟 Windows
- **文件名**: `Chaoxing.exe`
- **系统要求**: Windows 10/11 (64位)
- **使用方法**:
  1. 下载 `Chaoxing.exe`
  2. 右键选择"以管理员身份运行"
  3. 浏览器自动打开 http://localhost:5000

### 🍎 macOS
- **文件名**: `Chaoxing-macos`
- **系统要求**: macOS 10.15+ (Intel)
- **使用方法**:
  1. 下载 `Chaoxing-macos`
  2. 打开终端，运行: `chmod +x Chaoxing-macos`
  3. 运行: `./Chaoxing-macos`
  4. 如果提示安全警告，前往"系统偏好设置" > "安全性与隐私"允许运行
  5. 浏览器访问 http://localhost:5000

### 🤖 Android
- **文件名**: `chaoxing-signin.apk`
- **系统要求**: Android 7.0+
- **使用方法**:
  1. 下载 `chaoxing-signin.apk`
  2. 在手机上安装APK（需要允许安装未知来源应用）
  3. 打开应用即可使用

**注意**: Android版本是独立应用，不需要额外的服务器。所有功能都在应用内运行。

## 常见问题

### Windows

**Q: 提示"Windows已保护你的电脑"？**
A: 点击"更多信息"，然后选择"仍要运行"。这是因为应用没有数字签名。

**Q: 需要管理员权限吗？**
A: 推荐以管理员身份运行，但不是必须的。

### macOS

**Q: 提示"无法打开，因为无法验证开发者"？**
A:
1. 打开"系统偏好设置" > "安全性与隐私"
2. 点击"仍要打开"
3. 或者在终端运行: `xattr -cr Chaoxing-macos`

**Q: 如何在后台运行？**
A: 使用 `nohup ./Chaoxing-macos &` 命令

### Android

**Q: 无法安装APK？**
A:
1. 前往"设置" > "安全"
2. 允许"未知来源"或"安装未知应用"
3. 重新安装

**Q: 应用闪退？**
A: 确保Android版本在7.0以上，并给予应用必要的权限。

**Q: 后台运行会被杀掉？**
A:
1. 前往"设置" > "电池"
2. 将应用添加到"不受限制"列表
3. 关闭"电池优化"

## 技术说明

- **Windows/macOS**: 使用 pkg 将 Node.js 应用打包成独立可执行文件
- **Android**: 使用 Capacitor 将 Web 应用打包成原生 APK
